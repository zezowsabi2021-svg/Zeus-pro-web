# ZEUS PRO - محول الصور الإعلاني

تطبيق ويب احترافي لتحويل صور المنتجات إلى إعلانات جذابة وفيديوهات قصيرة لمنصات التواصل الاجتماعي.

## الميزات

- ✨ تحويل صور المنتجات إلى إعلانات احترافية
- 🎬 إنشاء فيديوهات قصيرة جذابة
- 🎨 تخصيص كامل للألوان والنصوص
- 📱 واجهة سهلة الاستخدام
- 🌍 دعم اللغة العربية

## التثبيت

```bash
npm install
```

## التشغيل

```bash
npm start
```

ثم افتح المتصفح على: http://localhost:3000

## النشر على Vercel

```bash
vercel
```

## الملفات الأساسية

- `server.js` - الخادم الرئيسي
- `public/index.html` - الواجهة الأمامية
- `package.json` - المكتبات المطلوبة
- `vercel.json` - إعدادات النشر
