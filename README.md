# ZEUS PRO - محول الصور الإعلاني

تطبيق ويب احترافي لتحويل صور المنتجات إلى إعلانات جذابة وفيديوهات قصيرة.

## المميزات

✅ **معالجة الصور:**
- إضافة نصوص إعلانية احترافية
- تخصيص ألوان النص
- تحكم بحجم الخط
- معاينة فورية

✅ **إنشاء الفيديو:**
- تحويل الصور إلى فيديوهات متحركة
- تحكم بمدة الفيديو
- جودة عالية (HD و Full HD)

✅ **التحميل والمشاركة:**
- تحميل الصور المعالجة
- تحميل الفيديوهات
- مشاركة مباشرة على وسائل التواصل

## المتطلبات

- Node.js 14+
- npm أو yarn

## التثبيت

```bash
# استنساخ المشروع
git clone <repo-url>
cd zeus-pro-web

# تثبيت المكتبات
npm install

# تشغيل التطبيق
npm start
```

التطبيق سيعمل على `http://localhost:3000`

## الاستخدام

1. **معالجة الصورة:**
   - اختر صورة المنتج
   - أضف النص الإعلاني
   - اختر اللون والحجم
   - اضغط "معالجة الصورة"
   - حمّل الصورة المعالجة

2. **إنشاء الفيديو:**
   - اختر صورة
   - حدد مدة الفيديو
   - اختر الجودة
   - اضغط "إنشاء الفيديو"
   - حمّل الفيديو

## النشر على Vercel

```bash
# تثبيت Vercel CLI
npm install -g vercel

# تسجيل الدخول
vercel login

# نشر المشروع
vercel
```
